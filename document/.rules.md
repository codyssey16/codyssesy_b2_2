# 문서 파일 작성하실 때 참고사항

## 문서 분류 document / images
* result 로 시작하는 문서는 워크플로우 개발 관련 분들이 작성하시는 문서입니다.
* qa 로 시작하는 문서는 테스트 담당하시는 분들이 작성하시는 문서입니다.
* project 로 시작하는 문서는 프로젝트 운영을 위해 작성하는 문서입니다.

## 문서에 무언가를 추가할 때
* image 는 images에 추가
* image가 아닌 다른 자원은 asset에 추가
